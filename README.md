# cod-ui

React component library. Wraps [material-ui](https://material-ui.com/).

## Features

- Uses `material-ui` as a foundational component library.
- Base [theme](https://material-ui.com/customization/themes/#themes).

## Installation

```sh
npm install --save cod-ui
```
